3. PRODUCT SCHEMA (on pages that list products):
Create a reusable utility function generateProductSchema(product) that returns:
{

  "@type": "Product",
  "name": product.name,
  "description": product.description,
  "brand": { "@type": "Brand", "name": product.brand },
  "offers": {
    "@type": "Offer",
    "price": product.price,
    "priceCurrency": "BDT",
    "availability": "https://schema.org/InStock",
    "seller": { "@type": "Organization", "name": "AI Premium Shop" },
    "url": "https://aipremiumshop.com/products"
  }
}

On /products page: wrap all product schemas in an ItemList.
On category pages (/ai-assistant, /ai-image, etc.): include only that category's products.
Render as <script type="application/ld+json"> in the page head.
